Guide de Lancement du Serveur YouTube Downloader

1. Préparation de l'environnement

Avant de lancer le serveur, assurez-vous que :

  • Python est installé sur votre système
  • yt-dlp est installé dans le bon répertoire
  • Vous avez accès au dossier du projet

2. Lancement du serveur

Suivez ces étapes pour lancer le serveur :

  1. Ouvrez l'Invite de commandes (cmd)
  2. Naviguez vers le dossier du projet :
    cd C:\Users\hp\youtube-downloader
  3. Lancez le serveur :
    python app.py

3. Vérification du bon fonctionnement

Si tout fonctionne correctement, vous devriez voir :

* Running on http://127.0.0.1:5000 * Debug mode: on

Pour accéder à l'application :

  1. Ouvrez votre navigateur
  2. Accédez à : http://127.0.0.1:5000

4. En cas de problème

Erreurs courantes et solutions :

  • Si vous voyez des erreurs concernant yt-dlp :
    • Vérifiez que le chemin dans le code correspond à votre installation
    • Assurez-vous que yt-dlp est correctement installé
  • Si le serveur ne démarre pas :
    • Vérifiez qu'aucun autre processus Python n'est en cours d'exécution
    • Essayez de fermer tous les terminaux et de redémarrer

5. Arrêt du serveur

Pour arrêter proprement le serveur :

  1. Dans le terminal, appuyez sur Ctrl+C
  2. Si nécessaire, forcez l'arrêt de tous les processus Python :
    taskkill /F /IM python3.13.exe

6. Redémarrage du serveur

Après chaque modification du code, vous devez :

  1. Arrêter le serveur (étape 5)
  2. Relancer le serveur (étape 2)

Important à retenir

  • Le serveur doit rester en cours d'exécution pendant que vous utilisez l'application
  • Ne fermez pas le terminal tant que vous utilisez l'application
  • En cas de problème, vérifiez les logs dans le terminal
